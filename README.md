Projeto de QA Manual – Sistema de Login

Descrição

Este projeto tem como objetivo demonstrar, na prática, a atuação de um QA (Quality Assurance) em testes manuais funcionais, por meio da criação de artefatos de teste aplicados a um sistema de login.

O projeto contém a documentação necessária para planejar, executar e evidenciar os testes realizados, incluindo cenários de teste, casos de teste e relatório de defeitos.


Objetivo

Validar o correto funcionamento da funcionalidade de login do sistema, garantindo que:

Usuários com credenciais válidas consigam acessar o sistema;

O sistema trate adequadamente entradas inválidas;

Mensagens de erro sejam exibidas corretamente.
